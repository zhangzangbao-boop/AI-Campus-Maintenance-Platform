## 核心场景覆盖

### 登录、权限和禁用用户

- `qiyun-user-service/AuthApiTests`：注册、登录、非法参数、匿名访问登录接口。
- `qiyun-user-service/AdminUserApiTests`：管理员创建/编辑/禁用/启用用户、重置密码、普通用户禁止管理员操作。
- `qiyun-user-service/JwtAuthenticationFilterTests`：禁用用户旧 Token 失效、无效 Token 拒绝。
- `qiyun-user-service/SecurityConfigApiTests`：公开路径和受保护路径权限规则。
- `qiyun-ops-service/OpsSecurityApiTests`：管理员接口、统计、反馈、知识库接口的认证与角色限制。
- `qiyun-repair-service/InternalRepairSecurityTests`：内部维修统计接口鉴权。

### 报修、派单、维修、学生确认、评价回访

- `qiyun-repair-service/TicketWorkflowTests`：工单创建、派单、维修状态流转、学生确认、评价。
- `qiyun-repair-service/TicketServiceTests`：工单查询、状态更新、删除、管理员处理。
- `qiyun-repair-service/StaffFeatureTests`：维修工任务列表、接单、详情、工作台。
- `qiyun-repair-service/RepairProcessRecordTests`：到场、过程、完成记录及图片处理。
- `qiyun-repair-service/TransferRequestTests`：转派申请与审批。
- `qiyun-ops-service/FeedbackManagementApiTests`：低分评价、回访筛选和回访状态更新。

### AI降级、RAG、规则配置

- `qiyun-ai-service/AiAnalyzeServiceTest`：本地规则分类、外部 AI 未配置时降级、配置化规则动态生效。
- `qiyun-ai-service/RagKnowledgeServiceTest`：Embedding/Chroma/AI 不可用时降级、相似知识召回。
- `qiyun-ai-service/InternalRagControllerTest`：内部 RAG 接口鉴权与参数。
- `qiyun-repair-service/AiAnalysisIntegrationServiceTests`：工单 AI 分析落库、空结果降级。
- `qiyun-repair-service/HistoricalRepairCaseServiceTests`：历史维修案例索引、检索、低相似度过滤、降级。
- `qiyun-ops-service/ConfigurableRepairRulesTests` 和 `qiyun-repair-service/RepairRuleConfigServiceTest`：规则配置读取、默认降级、动态生效、校验。

### 公告、通知、导出、备份和审计

- `qiyun-ops-service/AnnouncementApiTests`：公告创建、发布、可见性和权限。
- `qiyun-ops-service/InternalNotificationControllerTest`、`OpsCoreApiTests`：通知创建、读取、内部密钥鉴权、真实启用用户校验。
- `qiyun-ops-service/AdminExportApiTests`、`AdminExportServiceTests`：管理员导出、筛选条件透传、CSV转义、权限控制。
- `qiyun-ops-service/BackupApiTests`：备份列表/状态、权限、路径穿越阻断、备份/恢复失败响应。
- `qiyun-ops-service/AuditReliabilityTests`、`qiyun-repair-service/AuditEventPublisherTests`：关键操作审计、敏感字段脱敏、外部审计不可用时降级。

### 文件上传安全与非法参数

- `qiyun-repair-service/FileStorageServiceTests`：图片类型、扩展名、大小限制、路径安全。
- `qiyun-user-service/AuthApiTests`：登录/注册非法 JSON 与字段校验。
- `qiyun-repair-service/TicketServiceTests`、`TicketWorkflowTests`：非法状态、无权限访问、越权操作。
- `qiyun-ops-service/BackupApiTests`：`../` 和反斜杠路径穿越返回 400。
